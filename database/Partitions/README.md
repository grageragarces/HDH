# HDH Partitions Database

This directory contains partitioning results and method metadata.

## Structure

- **results/**: Main results database (`partitions.csv`)
- **methods/**: Method configurations and documentation
- **leaderboards/**: Pre-computed comparison views
- **configs/**: Test configurations and settings
- **scripts/**: Helper scripts for database operations

## Usage

See the [database documentation](https://grageragarces.github.io/HDH/database/) for details.
